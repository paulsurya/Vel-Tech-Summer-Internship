About the Project

A modern student registration system built with Flask, HTML, and CSS

Project Overview

What this system does and why it was built

The Student Registration System is a web-based application developed as part of the InnoTrack 2025 internship program at Vel Tech University. It allows administrative staff to register new students, view existing records, and manage student information across all departments efficiently. The system is built using Flask for backend routing, Jinja2 for template rendering, and custom HTML/CSS for a professional frontend interface.

Features

What the system currently supports

📋 Student Registration

Register new students with name, roll number, department, year, email, phone, gender, and address.

📊 Records Table

View all registered students in a clean, sortable table with department badges and year info.

🎨 Professional UI

Fully responsive design with a consistent color theme, hover effects, and styled components.

🔗 Flask Routing

Clean URL routing using Flask with render_template for all four pages of the application.
